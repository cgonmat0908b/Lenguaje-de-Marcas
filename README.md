# Lenguaje de marcas

## Carpeta que muestra el avance en la asignatura de Lenguaje de marcas

---

## Contenido del repositorio

### Ej 1. Favoritos
Primera página en HTML: una página sencilla en la que se usan algunos formatos, hipervínculos, listas ordenadas y desordenadas, colores, etc.

---

### Ej 2. Receta HTML
Segundo ejercicio de HTML.  
Se realiza una receta de un plato con imágenes, títulos (H1, H2), listas desordenadas con emojis como “numeración” de los ingredientes, lista ordenada con los pasos a seguir, enlaces a otras páginas y un breve vídeo de la receta.

---

### Ej 3. Historia
Tercer ejercicio de HTML.  
Se crea una historia usando algo de CSS para ordenar la página.  
La historia se divide en **introducción**, **nudo** y **desenlace**, y se incluye un índice para acceder a cada parte.

---

### Ej 4. Horario
Cuarto ejercicio de HTML.  
Se realiza una tabla con las etiquetas `table`, `tr`, `td`, `th`, etc., representando el horario del grupo.  
También se decora la página, se añade un pie de página y un icono a la pestaña del navegador.

---

### Ej 5. Formulario
Quinto ejercicio de HTML (aún en proceso).  
Consiste en la realización de un formulario en HTML.

---

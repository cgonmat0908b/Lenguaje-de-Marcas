Estilos CSS con Grid (styles.css) ```css * { margin: 0; padding: 0; box-sizing: border-box; font-family: Arial, sans-serif; } body { background-color: #f2f2f2; } /* Grid principal */ .grid-container { display: grid; grid-template-areas: "header" "main" "footer"; grid-template-rows: auto 1fr auto; min-height: 100vh; } .header { grid-area: header; background-color: #3f51b5; color: white; padding: 20px; text-align: center; } .menu { list-style: none; display: flex; justify-content: center; gap: 20px; margin-top: 10px; } .menu a { color: white; text-decoration: none; font-weight: bold; } .menu a:hover { color: #ffeb3b; } .main { grid-area: main; display: grid; grid-template-columns: 70% 30%; gap: 20px; padding: 20px; } .contenido { background-color: #ffffff; padding: 20px; border-radius: 10px; } .post { margin-bottom: 20px; } .post img { width: 100%; border-radius: 10px; margin: 10px 0; } .post h2 { color: #3f51b5; } .sidebar { background-color: #e8eaf6; padding: 20px; border-radius: 10px; } .sidebar h3 { color: #303f9f; margin-bottom: 10px; } .categorias ul { list-style: square; padding-left: 20px; } .footer { grid-area: footer; background-color: #303f9f; color: white; text-align: center; padding: 15px; } .footer a { color: white; margin: 0 10px; text-decoration: none; } .footer a:hover { color: #ffeb3b; } ``` --- ## 3. Explicación del diseño con CSS Grid ### a) Usando `grid-template-areas` **Grid principal (`.grid-container`):** ```css grid-template-areas: "header" "main" "footer"; ``` * Fila 1 → `header` * Fila 2 → `main` * Fila 3 → `footer` Cada sección ocupa todo el ancho de la página. **Grid del `main`:** ```css grid-template-columns: 70% 30%; ``` * Columna 1 (70%) → `.contenido` * Columna 2 (30%) → `.sidebar` --- ### b) Usando coordenadas del grid En el grid principal: * `header`: fila 1 / columna 1 * `main`: fila 2 / columna 1 * `footer`: fila 3 / columna 1 En el `main`: * `.contenido`: columna 1, ocupa el 70% del ancho * `.sidebar`: columna 2, ocupa el 30% del ancho Esto permite separar claramente el contenido principal del contenido secundario y mantener un diseño limpio y homogéneo. --- ✔️ Ejercicio completo: HTML semántico, CSS Grid, estilos homogéneos y pseudoclases aplicadas.
